# DABFY26 Dashboard

Interactive dashboard for analyzing Digital Access & Behavior Survey (DAB) data.

## Features

- **Demographics Overview**: Total respondents, gender, age, and regional breakdowns
- **Digital Behavior & Engagement**: Platform analysis and usage patterns
- **Internet Usage Behaviors**: Detailed behavioral analytics with demographic filters
- **Brand & Product Awareness**: Agricultural brands, Yetagon products, farming techniques, and NPS analysis

## Live Dashboard

[View Dashboard](https://dabfy26.streamlit.app) (link will be active after deployment)

## Local Development

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the dashboard:
```bash
streamlit run dashboard.py
```

## Data Files

- `cleaned.csv` - Main survey data
- `questions.csv` - Question metadata
